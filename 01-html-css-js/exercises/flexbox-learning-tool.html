‚úó Wrong 5. Mistakes to Avoid

‚ùå MISTAKE #1: Applying flex to the children instead of parent

Home
About
Contact
/* WRONG! ‚ùå */ .nav-item { display: flex; /* Don't put flex on children! */ justify-content: center; } /* CORRECT! ‚úì */ .header { display: flex; /* Put flex on parent! */ justify-content: center; }

Why it's wrong: Items won't move because flex controls the children of the element with display: flex, not the element itself.

‚ùå MISTAKE #2: Using justify-content without display: flex

About
Contact
/* WRONG! ‚ùå */ .header { justify-content: space-between; /* No effect without display: flex! */ } /* CORRECT! ‚úì */ .header { display: flex; /* MUST come first! */ justify-content: space-between; }

Why it's wrong: justify-content only works on flex containers. You MUST have display: flex first!

‚ùå MISTAKE #3: Not understanding when you have 1 vs multiple items

Key insight:
‚Ä¢ With 1 item: justify-content moves it left, center, or right
‚Ä¢ With 2+ items: justify-content controls spacing BETWEEN them
‚Ä¢ space-between with only 1 item does nothing!
