4. Custom Boolean Attributes (data-*)
Pro Tip: You can create your own boolean attributes using data-* attributes!
data-active="true"
Active State - Green Background
<div data-active="true">
data-active="false"
Inactive State - Gray Background
<div data-active="false">
5. CSS Boolean Attribute Selectors
CSS Magic: Style elements based on boolean attributes using [attribute] selectors!
[disabled] selector
input[disabled] { opacity: 0.5; }
[required] selector
input[required] { box-shadow: red; }
[data-active] selector
[data-active="true"] { background: green; }
Custom boolean styling works!
6. CSS Pseudo-class Boolean Selectors
Pseudo-classes: Special selectors that represent boolean states in CSS!
:checked
input:checked + span { color: green; }
:disabled vs :enabled
input:disabled { opacity: 0.5; }
:required vs :optional
input:required { box-shadow: red; }
:read-only vs :read-write
input:read-only { background: yellow; }
7. The :not() Boolean Trick (Inverse Selection)
:not() selector: Style the FALSE state by selecting "NOT true"!
:not(:disabled)
Style only enabled inputs
input:not(:disabled) { border: green; }
:not(:checked)
Style unchecked checkboxes
input:not(:checked) + span { color: gray; }
:not([hidden])
Style only visible elements
div:not([hidden]) { border: blue; }
Visible div with blue border
