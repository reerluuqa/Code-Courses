1. Main Header Container

/* The header creates a three-column layout */ header { display: flex; justify-content: space-between; align-items: center; }

Three flex children: Logo, Navigation, and Login Button

justify-content: space-between distributes these three elements with maximum space between them.

align-items: center ensures all three sections align vertically at their centers.

2. Logo Section (Left)

/* Logo stays on the left and doesn't grow */ .logo { display: flex; align-items: center; gap: 12px; flex: 0 0 auto; /* Don't grow or shrink */ }

flex: 0 0 auto means:

  • flex-grow: 0 - Logo won't expand to fill extra space
  • flex-shrink: 0 - Logo maintains its size
  • flex-basis: auto - Size determined by content

3. Navigation Section (Center) - THE KEY!

/* Nav grows to fill space and centers its items */ nav { display: flex; gap: 16px; align-items: center; flex: 1; /* Grows to fill available space */ justify-content: center; /* Centers items within */ }

flex: 1 - Navigation takes all available space between logo and button

justify-content: center - Navigation items are centered within that space

This creates the centered navigation effect!

4. Login Button (Right)

/* Button stays on the right and doesn't grow */ .login-button { flex: 0 0 auto; /* Don't grow or shrink */ }

flex: 0 0 auto - Button maintains its natural size and stays on the right

ðŸ“Š Complete Flex Strategy

This three-column pattern works because:

  • Logo (left): flex: 0 0 auto - stays fixed size on left
  • Nav (center): flex: 1 + justify-content: center - expands and centers content
  • Button (right): flex: 0 0 auto - stays fixed size on right
  • Header: justify-content: space-between - distributes all three evenly

Unlike Pattern 1 (which only has two children), Pattern 2 uses three flex children. The middle navigation section uses flex: 1 to grow and fill available space, then uses its own justify-content: center to center its links within that space.

Space Distribution:
Logo takes: ~150px (content width)
Button takes: ~120px (content width)
Nav takes: Remaining space (flex: 1)
Nav centers its content within that remaining space
