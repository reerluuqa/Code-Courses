HTML Display Guide

Understanding the difference between inline and block elements is fundamental to web development. This guide will help you master this concept with examples, exercises, and interactive demos.

The Core Concept

Every HTML element has a default display property that determines how it behaves in the layout.

BLOCK Elements

These elements take the full width available and start on a new line, like building blocks stacked vertically.

Examples: <div>, <h1>-<h6>, <p>, <section>

INLINE Elements These elements flow within text and don't start new lines.

Examples: <a>, <span>, <strong>, <img>

Memory Trick: Block elements are like paragraphs in a book (each starts on a new line). Inline elements are like words in a sentence (they flow together).

Code Examples

Block Elements in Action

<!-- Block elements stack vertically -->
<h1>My Website</h1>
<p>This is a paragraph.</p>
<div>
    <p>Another paragraph inside a div.</p>
</div>

Inline Elements in Action

<!-- Inline elements flow horizontally -->
<p>
    Click <a href="#">here</a> to
    learn <strong>more</strong> about
    <em>HTML</em>.
</p>

Mixed Example (Your Blog Navigation)

<!-- This is why your nav links are inline! -->
<nav>
    <a href="#">Home</a>
    <a href="#">Blog</a>
    <a href="#">About</a>
</nav>
<!-- <nav> is block, but <a> is inline -->

Practice Exercises

Exercise 1: Identify the Display Type

Look at this code and identify which elements will be block and which will be inline:

<article>
    <h2>My Article</h2>
    <p>This is <strong>important</strong> text.</p>
    <a href="#">Read more</a>
</article>

Block: <article>, <h2>, <p>

Inline: <strong>, <a>

Exercise 2: Fix the Layout

You want these navigation links to stack vertically (each on its own line). What's the simplest HTML change you can make without using CSS?

<nav>
    <a href="#">Home</a>
    <a href="#">About</a>
    <a href="#">Contact</a>
</nav>

Solution: Wrap each link in a block element like <li> inside a <ul> or <div>:

<nav>
    <div><a href="#">Home</a></div>
    <div><a href="#">About</a></div>
    <div><a href="#">Contact</a></div>
</nav>

Alternative: Use a list: <ul> with <li> items containing the links.

Comparison Table

Property Block Elements Inline Elements
Line Break Starts on a new line Continues on same line
Width Takes full available width Takes only needed width
Height/Width Can be set Cannot be set (except images)
Margin/Padding All sides respected Only horizontal respected fully
Contains Can contain inline & block Can only contain inline
