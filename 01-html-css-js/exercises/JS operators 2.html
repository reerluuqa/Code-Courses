What are Operators?

Operators are special symbols that perform operations on values and variables. Think of them as the verbs of JavaScript - they make things happen!

let result = 5 + 3; // + is the operator
let isValid = true && false; // && is the operator

Comparison Operators

Compare values and get a boolean result. These are essential for making decisions in your code!

== vs ===
Equality

== checks value only, === checks value AND type

5 == "5" → true
5 === "5" → false
!= vs !==
Inequality

!= loose, !== strict (recommended)

5 != "5" → false
5 !== "5" → true
> and <
Greater/Less Than

Compare numeric values

10 > 5 → true
3 < 3 → false
>= and <=
Greater/Less or Equal

Includes equality check

5 >= 5 → true
3 <= 2 → false
REAL WORLD

Practical Examples

// Age verification
let age = 21;
let canDrink = age >= 21;
console.log(canDrink); // true

let isTeenager = age >= 13 && age <= 19;
console.log(isTeenager); // false
// Login validation
let userInput = "admin";
let password = "secret123";

let isValid = userInput === "admin" &&
             password === "secret123";

console.log(isValid); // true
// Type checking pitfall
let value = "0";

console.log(value == 0); // true (coercion!)
console.log(value === 0); // false (different types)

// Always use === for predictable results!

Logical Operators

Combine multiple conditions to create complex logic. Think of these as "and", "or", and "not"!

&&
AND

Both must be true

true && true → true
true && false → false
||
OR

At least one must be true

true || false → true
false || false → false
!
NOT

Flips the boolean

!true → false
!false → true

Truth Tables

AND (&&)

A B A && B
true true true
true false false
false true false
false false false

OR (||)

A B A || B
true true true
true false true
false true true
false false false
REAL WORLD

Practical Examples

// Can user access admin panel?
let isLoggedIn = true;
let isAdmin = true;
let isBanned = false;

let canAccess = isLoggedIn && isAdmin && !isBanned;
console.log(canAccess); // true
// Weekend checker
let day = "Saturday";

let isWeekend = day === "Saturday" ||
               day === "Sunday";

console.log(isWeekend); // true
// Form validation
let email = "user@example.com";
let password = "";
let termsAccepted = true;

let canSubmit = email.length > 0 &&
               password.length >= 8 &&
               termsAccepted;

console.log(canSubmit); // false (password too short)

Assignment Operators

Shorthand ways to update variable values. These make your code cleaner and more readable!

=
Basic Assignment
let x = 5;
+= -=
Add/Subtract Assign
x += 3; // same as x = x + 3
x -= 2; // same as x = x - 2
*= /=
Multiply/Divide Assign
x *= 2; // same as x = x * 2
x /= 4; // same as x = x / 4
++ --
Increment/Decrement
x++; // same as x = x + 1
x--; // same as x = x - 1
REAL WORLD

Practical Examples

// Shopping cart
let total = 0;

total += 29.99; // Add item
total += 15.50; // Add another
total *= 1.08; // Apply 8% tax

console.log(total); // 49.13
// Game score
let score = 0;

score++; // Kill enemy: +1
score += 10; // Collect coin: +10
score -= 5; // Hit by enemy: -5
score *= 2; // Power-up: double!

console.log(score); // 12
// String concatenation
let message = "Hello";

message += ", ";
message += "World";
message += "!";

console.log(message); // "Hello, World!"
