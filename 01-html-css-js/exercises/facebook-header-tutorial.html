Flexbox Properties Cheat Sheet
Key Flexbox Properties Used
/* Container Properties (applied to parent) */ display: flex; /* Activates flexbox */ justify-content: space-between; /* Horizontal distribution */ align-items: center; /* Vertical alignment */ gap: 8px; /* Space between children */ /* Child Properties (applied to children) */ flex: 1; /* Grow to fill space */ flex: 0 0 auto; /* Don't grow or shrink */

üéØ When to Use Each:

  • justify-content: space-between - Push items to edges with even spacing
  • justify-content: center - Center items horizontally
  • align-items: center - Center items vertically
  • gap - Modern way to add spacing (no margin needed!)
  • flex: 1 - Make element grow to fill available space
