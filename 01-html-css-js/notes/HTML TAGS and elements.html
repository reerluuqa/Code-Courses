üåü What is HTML?

HTML stands for HyperText Markup Language. Think of it as the skeleton or building blocks of a website. Just like LEGO blocks, you use HTML to build the structure of your webpage!

üß∏ Simple Explanation: Imagine you're building a house with toy blocks:
  • HTML is like the blocks themselves (walls, doors, windows)
  • CSS is like the paint and decorations (colors, styles, design)
  • JavaScript is like the electricity (making things move and interactive)

üè∑Ô∏è What is a TAG?

A tag is like a label or instruction that tells the browser what kind of content you're creating. Tags are wrapped in angle brackets < >.

<tagname> Your content here </tagname>
üéØ Important Parts of a Tag:
  • Opening Tag: <p> - This starts the instruction
  • Content: The text or other elements inside
  • Closing Tag: </p> - This ends the instruction (notice the /)

Types of Tags:

1. Paired Tags (Most Common)

These have an opening and closing tag:

<p>This is a paragraph.</p>
<h1>This is a heading</h1>

2. Self-Closing Tags (Special Ones)

These don't need a closing tag because they don't contain content:

<img src="picture.jpg" alt="A picture">
<br>
<hr>

üß© What is an ELEMENT?

An element is the COMPLETE package: the opening tag + content + closing tag all together!

<h1> Hello World! </h1>

üëÜ This whole thing is one element
üí° Easy Way to Remember:
  • Tag: Just the label part ‚Üí <p> or </p>
  • Element: Everything together ‚Üí <p>Hello!</p>

üìö Important Elements You Must Know

1. Heading Elements (h1 to h6)

These create titles and headings. <h1> is the biggest and most important, <h6> is the smallest.

Example: Headings
<h1>Main Title (Biggest)</h1>
<h2>Section Title</h2>
<h3>Subsection Title</h3>
<h4>Smaller Heading</h4>
<h5>Even Smaller</h5>
<h6>Smallest Heading</h6>
How it looks:

Main Title (Biggest)

Section Title

Subsection Title

Smaller Heading

Even Smaller
Smallest Heading

2. Paragraph Element (<p>)

Used for regular text paragraphs.

Example: Paragraph
<p>This is a paragraph of text. It can be as long as you want!</p>
<p>This is another paragraph.</p>
How it looks:

This is a paragraph of text. It can be as long as you want!

This is another paragraph.

3. Link Element (<a>)

Creates clickable links to other pages or websites.

Example: Links
<a href="https://www.google.com">Click here to go to Google</a>
How it looks:
Click here to go to Google
üîë About "href": The href is called an attribute. Attributes give extra information about an element. For links, href tells the browser where to go when clicked.

4. Image Element (<img>)

Displays images on your webpage. This is a self-closing tag!

Example: Image
<img src="cat.jpg" alt="A cute cat">
üñºÔ∏è Image Attributes:
  • src - The source/location of the image file
  • alt - Description of the image (important for accessibility!)

5. List Elements

There are two types of lists:

Unordered List (Bullet Points)

Example: Unordered List
<ul>
  <li>Apples</li>
  <li>Bananas</li>
  <li>Oranges</li>
</ul>
How it looks:
  • Apples
  • Bananas
  • Oranges

Ordered List (Numbered)

Example: Ordered List
<ol>
  <li>First step</li>
  <li>Second step</li>
  <li>Third step</li>
</ol>
How it looks:
  1. First step
  2. Second step
  3. Third step

6. Division Element (<div>)

A container to group other elements together. Like a box to organize things!

Example: Div Container
<div>
  <h2>My Section</h2>
  <p>This is inside a div container.</p>
</div>
How it looks:

My Section

This is inside a div container.

7. Span Element (<span>)

Like a mini-container for small pieces of text inside other elements.

Example: Span
<p>I love <span style="color: red;">red apples</span> and oranges.</p>
How it looks:

I love red apples and oranges.

8. Button Element (<button>)

Creates a clickable button.

Example: Button
<button>Click Me!</button>

10. Strong and Emphasis Elements

Example: Bold and Italic

üèóÔ∏è Basic HTML Document Structure

Every HTML page has a basic structure, like a recipe has ingredients and steps:

Complete HTML Page Template
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>My First Webpage</title>
</head>
<body>
    <h1>Welcome to My Website!</h1>
    <p>This is my first webpage.</p>
</body>
</html>
üìù What Each Part Means:
  • <!DOCTYPE html> - Tells the browser this is an HTML5 document
  • <html> - The root element, contains everything
  • <head> - Contains information about the page (not visible on the page)
  • <title> - The title shown in the browser tab
  • <body> - Contains everything you see on the page

üé® What is CSS?

CSS stands for Cascading Style Sheets. It's what makes your HTML look pretty! Without CSS, every website would look plain and boring.

How to Add CSS to HTML

Method 1: Inline CSS (Inside the Tag)

Example: Inline CSS
<p style="color: blue; font-size: 20px;">This text is blue and big!</p>
How it looks:

This text is blue and big!

Method 2: Internal CSS (In the Head Section)

Example: Internal CSS
<head>
    <style>
        p {
            color: green;
            font-size: 18px;
        }
    </style>
</head>
<body>
    <p>This paragraph will be green!</p>
</body>

Common CSS Properties

color

Changes text color

color: red;

background-color

Changes background color

background-color: yellow;

font-size

Changes text size

font-size: 20px;

font-weight

Makes text bold

font-weight: bold;

text-align

Aligns text

text-align: center;

padding

padding: 10px;

margin

Space outside element

margin: 10px;

border

Adds a border

border: 2px solid black;
Example: Styled Element
<div style="
    background-color: lightblue;

    border-radius: 10px;

    <h2 style="color: navy;">My Styled Box</h2>

How it looks:

My Styled Box

Look how pretty this is!

‚ö†Ô∏è Common Mistakes to Avoid

‚ùå Mistake #1: Forgetting to Close Tags 
Wrong: <p>Hello World!
Correct: <p>Hello World!</p>
‚ùå Mistake #2: Misspelling Tag Names 
Wrong: <paragraf>Text</paragraf>
Correct: <p>Text</p>
‚ùå Mistake #3: Not Using Lowercase for Tags 
Wrong: <P>Text</P>
Correct: <p>Text</p>

Always use lowercase for HTML tags!

‚ùå Mistake #4: Nesting Tags Incorrectly 
Wrong: <strong><em>Text</strong></em>
Correct: <strong><em>Text</em></strong>

Tags must close in the opposite order they were opened!

‚ùå Mistake #5: Forgetting Alt Text for Images 
Wrong: <img src="cat.jpg">
Correct: <img src="cat.jpg" alt="A cute orange cat">

Always add alt text to describe your images!

‚ùå Mistake #6: Using Too Many <br> Tags 
Wrong: 
<p>Line 1<br><br><br><br>Line 2</p>

Correct: 
<p>Line 1</p>
<p>Line 2</p>

Use separate paragraphs or CSS margins instead!

‚ú® Best Practices (Good Habits to Learn)

‚úÖ 1. Indent Your Code

Make your code easy to read by indenting nested elements:

<div>
    <h1>Title</h1>
    <p>Paragraph</p>
</div>
‚úÖ 2. Use Meaningful Names

When adding classes or IDs, use names that describe what the element is:

Good: <div class="header">
Bad: <div class="box1">
‚úÖ 3. Comment Your Code

Add notes to help you remember what parts of your code do:

<!-- This is a comment -->
<!-- Navigation menu starts here -->
<nav>...</nav>
‚úÖ 4. One Main Heading Per Page

Use only ONE <h1> tag per page (usually the main title).

‚úÖ 5. Test in Different Browsers

Your webpage might look different in Chrome, Firefox, Safari, etc. Always test!
