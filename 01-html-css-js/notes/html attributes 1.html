üö® Common Mistakes to Avoid

‚ùå Mistake #1: Curly Quotes (Copy-Paste Killer)

When copying from Word or Google Docs:

<a href=‚Äùhttps://site.com‚Äù>Wrong</a>

‚úÖ Fix: Type quotes manually in your code editor: " "

‚ùå Mistake #2: Boolean Confusion

These do NOT work to enable an element:

<input disabled="false"> ‚ùå Still disabled!
<input disabled="no"> ‚ùå Still disabled!
<input disabled="0"> ‚ùå Still disabled!

‚úÖ Fix: Remove the attribute completely to enable: <input>

‚ùå Mistake #3: Unquoted Values with Spaces

Browser thinks these are 3 separate attributes!

‚úÖ Fix: <div class="big red box">

üí° Pro Tips

  • Always use straight quotes " not curly ‚Äù
  • Attributes are lowercase: class not CLASS
  • No spaces around = : class="name" not class = "name"
  • Order doesn't matter: src before or after alt is fine
