# Databases

**Course 05:** Data Storage & Management

## Overview

Master both SQL and NoSQL databases. Learn data modeling, complex queries, indexing, and how to choose the right database for your application.

## Topics Covered

- SQL Databases (PostgreSQL/MySQL)
- NoSQL Databases (MongoDB)
- Database Design
- Query Optimization
- Indexing
- Transactions & ACID
